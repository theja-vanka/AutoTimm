# Getting Started Examples

Start your AutoTimm journey with these fundamental examples.

## Examples

### Classification Basics

- **`classify_cifar10.py`** - Train ResNet-18 on CIFAR-10 with MetricManager
- **`classify_custom_folder.py`** - Train on custom ImageFolder dataset with W&B logging
- **`vit_finetuning.py`** - Two-phase Vision Transformer fine-tuning

## Quick Start

```bash
# Run a basic example
python getting_started/classify_cifar10.py

# Train on your custom dataset
python getting_started/classify_custom_folder.py
```

- Creating and training models
- Using metrics and logging
- Fine-tuning pretrained models
